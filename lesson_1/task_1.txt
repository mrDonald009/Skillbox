Задача 
Цель: научиться пользоваться тестовой средой — запускать простые SQL-запросы:
•	для получения всех данных из таблицы;
•	для получения первых нескольких строк;
•	в различных базах данных.

Обратите внимание
Для успешной сдачи домашнего задания необходимо выполнить задания 1, 2, 3, 4 и отправить их на проверку преподавателю.
Задание 5 относится к категории повышенной сложности. Его выполнение желательно, но необязательно. Прежде чем приступить к выполнению, прочтите эту инструкцию до конца.

Задание 1
В базе данных, которая расположена в тестовой среде, содержатся следующие таблицы: skill_cities, skill_managers и skill_product_category. 
1.	Для каждой таблицы напишите SQL-запрос, который получает все данные таблицы — все колонки и все строки.
2.	Выполните каждый SQL-запрос в тестовой среде.

Задание 2
В базе данных Microsoft SQL Server, которая расположена в тестовой среде, содержатся следующие таблицы: skill_cities, skill_managers и skill_product_category. 
1.	Для каждой таблицы напишите SQL-запрос, который получает только первые пять строк таблицы.
2.	Выполните каждый SQL-запрос в тестовой среде.

Задание 3
В базе данных MySQL, которая расположена в тестовой среде, содержатся следующие таблицы: skill_cities, skill_managers и skill_product_category. 
1.	Для каждой таблицы напишите SQL-запрос, который получает только первые пять строк таблицы.
2.	Выполните каждый SQL-запрос в тестовой среде.

Задание 4
В базе данных, которая расположена в тестовой среде, содержится таблица со списком сотрудников отдела продаж.
Разработчик базы данных в отпуске и не помнит точного названия таблицы, он сумел прислать список, в котором есть одно верное название:
dispatchers, distributors, suppliers, sellers, exporters
1.	Напишите и выполните SQL-запрос, который получает все строки из таблицы со списком сотрудников.
2.	Напишите и выполните SQL-запрос, который получает первые четыре строки из таблицы со списком сотрудников.
3.	Укажите в пояснениях к заданию, сколько строк содержится в таблице.

Задание 5
В базе данных, которая расположена в тестовой среде, содержится таблица, название которой есть в данном списке:
dispatchers, distributors, suppliers, sellers, exporters
Используя оператор SELECT, выясните:
1.	Как называется таблица.
2.	Какие колонки есть в таблице.
3.	Сколько строк с данными содержится в таблице.
Сделайте скриншот и напишите ответы на данные вопросы в комментариях.

Общие рекомендации:
•	Тестовая среда позволяет выполнять одновременно несколько SQL-запросов. Ответы можно вводить в поля помеченные: ‘--enter sql’.
•	После заполнения SQL-запросов нужно запустить их на выполнение, нажав ‘run’, и проанализировать полученный результат напротив каждого SQL-запроса.
•	Во всех заданиях обращайте внимание на то, в среде какой базы данных должны выполняться запросы.
Для всех задач в качестве результата пришлите скриншоты тестовой среды, на которых видны написанные вами SQL-запросы и таблицы с результатами. Также рекомендуем приложить свои пояснения по тому, как вы выполняли конкретное задание.

Критерии оценки: 
«Зачёт» — в тестовой среде выполнены SQL-запросы, выведены данные, которые содержатся в таблицах. Зачёт ставится при успешном выполнении заданий 1, 2, 3, 4.
Задание 5 относится к категории повышенной сложности. Его выполнение желательно, но необязательно для получения зачёта.
«На доработку» — выполнены не все SQL-запросы, не получены данные или тестовая среда выдает ошибку, связанную с синтаксисом SQL-запроса.
